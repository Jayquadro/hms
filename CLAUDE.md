# Istruzioni per Claude

## Regole generali

- **Non aggiornare mai le librerie** (versioni nei `package.json`) a meno che non venga esplicitamente richiesto dall'utente.
- Non modificare le dipendenze esistenti nei file `package.json` salvo istruzione esplicita.
- Se una funzionalità richiede una nuova dipendenza, chiedere conferma prima di aggiungerla.
